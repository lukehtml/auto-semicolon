# Auto-Semicolon

#### Automatically adds a semicolon when you press Enter

> Tired of forgetting semicolons? This tool automatically adds them for you while typing, with a pause toggle when needed.

---

## How to Use

1. **Clone the repository**  
   git clone https://github.com/lukehtml/auto-semicolon.git

2. **Install Python** (if you haven’t already)  
   Download from https://www.python.org/

3. **Install dependencies**  
   python -m pip install -r requirements.txt

4. **Run the program**  
   python main.py

And that’s it! You won’t forget semicolons anymore.

---

## Controls

- **Enter**  
  Automatically inserts a semicolon when pressed

- **Right Shift**  
  Toggles pause / unpause

---

## Features

- Automatically inserts a semicolon on Enter  
- Pause and resume typing instantly  
- Lightweight and easy to use  
- Perfect for custom typing macros
