# Auto-Semicolon
#### Automatically adds a semicolon when you press space

> Tired of forgetting semicolons? This tool automatically adds them for you while typing.

---

## How to Use

1. **Clone the repository**  
   ```
   git clone https://github.com/lukehtml/auto-semicolon.git
   ```

2. **Install Python** (if you haven’t already)  
   Download from [python.org](https://www.python.org/)

3. **Install dependencies**  
   ```
   python -m pip install -r requirements.txt
   ```

4. **Run the program**  
   ```
   python main.py
   ```

And that’s it! You won't forget those semicolons anymore!

---

## Features

- Automatically inserts a semicolon when you press space  
- Lightweight and easy to use  
- Perfect for your own custom projects
